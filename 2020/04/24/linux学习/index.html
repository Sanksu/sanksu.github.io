一些简单的命令

日期与时间

date

日历

cal

格式

cal [month] [year]

计算器

bc

apt包管理

安装

apt-get install [包名] (-y)

移除

apt-get remove [包名] (-y)

更新源

apt-get update

更新包

apt-get upgrade

定位

cd [路径]

列出文件

ls

查看当前路径

pwd

删除

rm

参数

  • -i > 逐一询问确认
  • -rf > 删除目录中所有文件，并且不用确认

创建

mkdir [创建的文件名]

复制

cp [复制的文件路径] [粘贴的文件路径]

权限

chmod 777 [文件名] -c（或：chmod u=rwx,g=rwx,o=rwx t.log -c) # 授予文件777的权限
